AntiHax does exactly what it says in the name, it stops hacks. There are many hacks that it stops, which are listed below (we need more)

Hacks Blocked:
  - Reach
  - Walk on Water
  - Sprinting Without Food
  - Flying
  - Hitting Yourself
  - Cursing (Chat)
  - Forcefield
  - AutoEnchant
  - FullBright
  - DropInventory
  - Nuker
  - HighJump/Step/Spider

Mods Blocked:
  - CJB
  - REI's Minimap
  - Zombe's Modpack
